ClassBlockedArray<T, int logBlockSize = 2>

Implements a generic 2D array of values. Rather than storing values so that entire rows are contiguous in memory, the data is ordered in memory using a blocked memory layout. The array is subdivided into square blocks of BLOCK_SIZE (in power of 2).

logBlockSize is specified the logarithm (base 2) of the block size.
